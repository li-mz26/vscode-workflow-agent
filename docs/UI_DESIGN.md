# VSCode Workflow Agent - UI/UX 设计文档

## 1. 设计原则

### 1.1 核心原则
- **清晰性**：每个元素的目的明确，不造成认知负担
- **效率**：常用操作不超过 2 次点击
- **一致性**：遵循 VSCode 设计规范，与原生体验无缝融合
- **反馈**：操作有即时反馈，状态变化清晰可见

### 1.2 视觉风格
- **设计语言**：现代、简洁、专业
- **色彩系统**：继承 VSCode 主题色，节点使用语义化颜色
- **字体**：使用 VSCode 默认字体栈
- **间距**：8px 网格系统

---

## 2. 布局架构

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│ VSCode 标题栏                                                │
├──────────┬──────────────────────────────────────────────────┤
│          │  ┌────────────────────────────────────────────┐  │
│  活动栏   │  │  工具栏 (Toolbar)                           │  │
│          │  ├────────────────────────────────────────────┤  │
│  📁      │  │                                            │  │
│  🔀      │  │                                            │  │
│  ⚙️      │  │         工作流画布 (Canvas)                 │  │
│          │  │                                            │  │
│          │  │                                            │  │
│          │  │                                            │  │
│          │  └────────────────────────────────────────────┘  │
│          │  ┌────────────────────────────────────────────┐  │
│          │  │  底部面板 (Bottom Panel)                    │  │
│          │  │  [日志] [变量] [执行历史] [控制台]           │  │
│          │  └────────────────────────────────────────────┘  │
├──────────┼──────────────────────────────────────────────────┤
│  侧边栏   │  属性面板 / 节点面板 (根据上下文切换)            │
└──────────┴──────────────────────────────────────────────────┘
```

### 2.2 区域详解

#### 活动栏 (Activity Bar)
新增 Workflow 图标，点击后：
- 显示工作流资源管理器（文件树）
- 显示节点面板（可拖拽）

#### 画布区域 (Canvas)
- 占据主要空间
- 背景：点阵网格（可配置密度/颜色）
- 支持缩放（10% - 200%）
- 迷你地图（右下角，可收起）

#### 侧边栏 (Sidebar)
上下文敏感面板：
- **无选中**：显示节点库（Node Library）
- **选中节点**：显示属性编辑器（Property Editor）
- **多选**：显示批量操作面板

#### 底部面板 (Bottom Panel)
标签页切换：
- **日志 (Logs)**：执行日志输出
- **变量 (Variables)**：当前执行上下文变量
- **历史 (History)**：执行历史记录
- **控制台 (Console)**：REPL 调试

## 3. 节点设计

### 3.1 节点外观

```
┌─────────────────────────────────┐
│ ●  节点名称              [状态] │  ← 头部 (Header)
├─────────────────────────────────┤
│                                 │
│  [输入端口]  内容区域  [输出端口] │  ← 主体 (Body)
│                                 │
├─────────────────────────────────┤
│  📄 描述文本...                 │  ← 底部 (Footer)
└─────────────────────────────────┘
```

**尺寸规范**：
- 最小宽度：160px
- 标准宽度：200px
- 高度：自适应（最小 80px）
- 圆角：8px
- 阴影：选中时显示发光效果

### 3.2 节点类型颜色

| 类型 | 主色 | 用途 |
|------|------|------|
| Start | `#4CAF50` (绿) | 工作流开始 |
| End | `#F44336` (红) | 工作流结束 |
| Code | `#2196F3` (蓝) | 代码执行 |
| LLM | `#9C27B0` (紫) | AI 模型调用 |
| Switch | `#FF9800` (橙) | 条件分支 |
| Parallel | `#00BCD4` (青) | 并行执行 |
| Merge | `#795548` (棕) | 汇聚节点 |

### 3.3 节点状态样式

- **默认**：正常边框
- **选中**：蓝色发光边框 + 阴影
- **运行中**：脉冲动画 + 旋转图标
- **成功**：绿色边框 + 对勾图标
- **失败**：红色边框 + 错误图标
- **警告**：黄色边框 + 警告图标

### 3.4 端口设计

```
输入端口                    输出端口
   ●───                      ───●
   ▲                         ▲
   │                         │
  圆形                        圆形
直径: 12px                 直径: 12px
颜色: 类型对应              颜色: 类型对应
```

**端口类型标识**：
- 数据端口：实心圆
- 控制流端口：空心圆
- 可连接时：放大 + 发光

## 4. 交互设计

### 4.1 画布交互

| 操作 | 触发方式 | 反馈 |
|------|---------|------|
| 平移 | 空格+拖拽 / 中键拖拽 | 手型光标 |
| 缩放 | 滚轮 / Cmd+滚轮 | 缩放指示器 |
| 框选 | Shift+拖拽 | 半透明选框 |
| 适应视图 | 双击空白 / 点击工具栏按钮 | 平滑动画过渡 |
| 显示/隐藏网格 | 工具栏按钮 | 即时切换 |

### 4.2 节点交互

| 操作 | 触发方式 | 反馈 |
|------|---------|------|
| 创建 | 从节点库拖拽到画布 | 拖拽预览 + 放置动画 |
| 移动 | 拖拽节点 | 实时位置更新 |
| 选中 | 单击 | 高亮边框 |
| 多选 | Cmd+单击 / 框选 | 批量高亮 |
| 删除 | Delete / 右键菜单 | 淡出动画 |
| 复制 | Cmd+C | 节点闪烁 |
| 粘贴 | Cmd+V | 新节点出现在原位置偏移处 |
| 编辑 | 双击 / 选中后侧边栏编辑 | 侧边栏滑入 |

### 4.3 连接交互

| 操作 | 触发方式 | 反馈 |
|------|---------|------|
| 创建连接 | 从输出端口拖拽到输入端口 | 拖拽线跟随鼠标 |
| 删除连接 | 单击选中 + Delete | 线消失动画 |
| 重新路由 | 拖拽线上的控制点 | 实时更新路径 |
| 无效连接 | 拖拽到不兼容端口 | 红色禁止图标 |

### 4.4 右键菜单

**画布右键**：
```
┌─────────────────────┐
│  添加节点    ▶      │
│  粘贴              │

│  适应视图           │
│  显示网格    [✓]    │
│  ─────────────────  │
│  撤销              │
│  重做              │
└─────────────────────┘
```

**节点右键**：
```
┌─────────────────────┐
│  编辑属性           │
│  复制              │
│  删除              │
│  ─────────────────  │
│  禁用/启用          │
│  设为断点    [ ]    │
└─────────────────────┘
```

## 6. 动画与过渡

### 6.1 动画时长
- 快速反馈：150ms
- 标准过渡：250ms
- 复杂动画：350ms

### 6.2 动画效果

| 场景 | 动画 | 时长 |
|------|------|------|
| 节点创建 | 缩放弹出 | 250ms |
| 节点删除 | 淡出 + 缩小 | 200ms |
| 节点选中 | 边框发光 | 150ms |
| 连接创建 | 线条绘制 | 200ms |
| 面板切换 | 滑动 | 250ms |
| 执行中 | 脉冲 + 旋转 | 循环 |
| 执行完成 | 绿色闪烁 | 300ms |
| 错误 | 红色抖动 | 300ms |

### 6.3 缓动函数
- 标准：`cubic-bezier(0.4, 0.0, 0.2, 1)`
- 进入：`cubic-bezier(0.0, 0.0, 0.2, 1)`
- 退出：`cubic-bezier(0.4, 0.0, 1, 1)`
- 弹性：`cubic-bezier(0.68, -0.55, 0.265, 1.55)`
